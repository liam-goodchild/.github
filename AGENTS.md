# AGENTS.md

This file provides guidance to Codex CLI and other AI coding agents when working with code in this repository.

## Repository Purpose

Default community health repository for the `liam-goodchild` GitHub account. Files under `.github/` provide shared issue templates, pull request templates, and account-level defaults for repositories that do not define their own.

## Structure

- `.github/ISSUE_TEMPLATE/` — shared issue templates and template chooser configuration.
- `.github/PULL_REQUEST_TEMPLATE/` — shared pull request templates for bug fixes, features, and maintenance.
- `.github/workflows/tag.yml` — shared tagging workflow.
- `README.md` — short repository description.

## Working Guidelines

- Keep templates generic enough to apply across all repositories in the account.
- Prefer small Markdown-only changes unless intentionally updating shared automation.
- Preserve existing template frontmatter and form field names unless intentionally changing the user-facing workflow.
